# Node Auth Server

You can access the API as well as above code freely and use it.

### BaseUrl:

`https://sathi-auth.onrender.com`

### Tech Stack:

- Node.js
- Expres.js
- Mongo Atlas
- Bcryptjs

### How to Run:

1. To Run install the dependencies

```bash
  npm install
```

2. To run the server locally

```bash
  npm run dev
```

```bash
  npm start
```
